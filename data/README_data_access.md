# Data Access Statement

The dataset used in this project is hyperspectral egg data obtained from external public source.

The dataset is included in this repository (spectra and reference parameters.csv).

## Access Instructions
- The data originates from publicly available research datasets.
- Access method: download from Illinois Data Bank (public).
- Song, Di (2025): Data for egg hyperspectral image. University of Illinois Urbana-Champaign. https://doi.org/10.13012/B2IDB-8141497_V1

## Data Ownership
The dataset is owned by Di Song. This project does not claim ownership.
